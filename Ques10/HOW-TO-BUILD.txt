Software used: apache-ant-1.10.1
OS used: Windows10

The project can be built using ANT on Windows10.

In the command prompt, change the current directory to:
ppl-assignment-LIT2015017/Ques10

Create a folder "classes" if not already present and copy the text files "girl.txt", "boy.txt", "essential.txt", "utility.txt", "luxury.txt".
A copy of these text files should also be present in the directory "Ques10". 

The build.xml file is given in the folder. Just type "ant" in the command prompt.The .class files of all the java files will be built in the folder "classes".
